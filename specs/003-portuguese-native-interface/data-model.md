## Formatting Conventions

### Date Format Rules

**Pattern for dates**:
```
dd de mmmm de yyyy
Example: 12 de abril de 2026
```

**Implementation**:
```typescript
const months = ['janeiro', 'fevereiro', 'março', 'abril', 'maio', 'junho',
                'julho', 'agosto', 'setembro', 'outubro', 'novembro', 'dezembro'];

function formatDatePortuguese(date: Date): string {
  const day = date.getDate();
  const month = months[date.getMonth()];
  const year = date.getFullYear();
  return `${day} de ${month} de ${year}`;
}
```

### Time Ago Format Rules

**Pattern for elapsed time**:
```
há X unidade
Examples: "há 2 horas", "há 5 dias"
```

**Implementation**:
```typescript
function formatTimeAgoPortuguese(hours: number): string {
  if (hours < 24) {
    return `há ${hours} horas`;
  } else {
    const days = Math.floor(hours / 24);
    return `há ${days} dias`;
  }
}
```

### Number Formatting Rules

**Decimal separator**: Comma (,)  
**Thousands separator**: Period (.)  
**Pattern**: `#.###,##`  

**Example**:
```
English: 1234.56 → Portuguese: 1.234,56
English: 1000000 → Portuguese: 1.000.000
```

**Implementation** (if formatting is displayed):
```typescript
function formatNumberPortuguese(num: number): string {
  return num.toLocaleString('pt-BR', {
    minimumFractionDigits: 0,
    maximumFractionDigits: 2
  });
}
```
